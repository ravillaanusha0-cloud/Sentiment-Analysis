# Sentiment-Analysis

A Python console application that analyzes the sentiment of text and classifies it as
Positive, Negative, or Neutral using NLP techniques.

## Features
- Sentiment analysis of user input text
- Positive / Negative / Neutral classification
- Simple and fast execution

## Technologies
- Python
- TextBlob (NLP)
